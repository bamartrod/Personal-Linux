1. Editor de código Neovim (nvim)
   * Archivo: ~/.config/nvim/init.lua
   * Contenido: Define plugins (como exploradores de archivos, autocompletado), atajos de teclado y el esquema de colores.
   * Estado: Estructura lazy.nvim para configuración modular y profesional.

  2. Gestor de ventanas i3 (i3)
   * Archivo: ~/.config/i3/config
   * Contenido: Define cómo se abren las ventanas, los atajos de teclado y la apariencia de las barras de estado

  3. Servidor gráfico Xorg
   * Directorio: /etc/X11/xorg.conf.d/
   * Contenido: Configuraciones de monitores y aceleración de video.

  4. Terminal (Bash)
   * Archivo: ~.bashrc
   * Contenido:
       * Aliases: Atajos para comandos largos
       * PS1: Define cómo se ve el "prompt"
       * PATH
